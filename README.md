学生签到/管理系统
===============

## 项目介绍
基于Vue + Thinkphp的前后端分离设计.

*演示地址：http://qiandao2019.gz01.bdysite.com/
*用户名: admin
*密码：123456


## 主要使用技术
* 后端框架：ThinkPHP 5.0.x/ThinkPHP 5.1.x
* 前端MVVM框架：Vue.JS 2.x
* 开发工作流：Webpack 1.x
* 路由：Vue-Router 2.x


详细开发文档参考 [ThinkPHP5完全开发手册](http://www.kancloud.cn/manual/thinkphp5)

## 后台系统功能

* 登录、退出登录
* 修改密码、记住密码
* 菜单管理
* 角色管理
* 学校管理
* 院系管理
* 作息时间管理
* 用户管理
* 留言管理
